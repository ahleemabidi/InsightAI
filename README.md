## <span style="color:#FF5733;">Description</span>

**InsightAI** est une IA prédictive conçue pour évaluer l'impact des commits sur le reste du code et pour prédire la probabilité de régression. Grâce à des algorithmes avancés de machine learning et à l'analyse du code source, InsightAI aide les développeurs à identifier les risques potentiels avant l'intégration des modifications.

## <span style="color:#FF5733;">Fonctionnalités</span>

- <img src="https://img.icons8.com/color/48/000000/artificial-intelligence.png" alt="AI" width="24"/> **Prédiction de l'impact des commits** : Analyse chaque commit pour évaluer son impact potentiel sur le reste du codebase.
- <img src="https://img.icons8.com/color/48/000000/bug.png" alt="Bug" width="24"/> **Détection des régressions** : Prédiction de la probabilité de régression pour chaque modification apportée.
- <img src="https://img.icons8.com/color/48/000000/report-file.png" alt="Report" width="24"/> **Rapports détaillés** : Génération de rapports sur les risques potentiels liés à chaque commit.
- <img src="https://img.icons8.com/color/48/000000/continuous-integration.png" alt="CI/CD" width="24"/> **Intégration avec CI/CD** : Compatible avec les principaux systèmes d'intégration continue pour automatiser l'analyse des commits.
